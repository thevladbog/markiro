# Design Brief 05 — Landing Page

> Final stage. Office mode + full brand. Desktop 1440 first, fully responsive
> to mobile (buyers will open it from a phone after a call). RU primary,
> EN version planned. Light theme primary; dark optional.

## Goal

Generate **demo/pilot requests** from small production plants. This is not
card-payment self-service: the decision maker is an owner/director who talks
to us first. Primary CTA — «Запросить демо» (request a demo) + visible phone
number. Secondary CTA — short "how it works" video placeholder.

## Audience & tone

Owner of a small plant (brewery, water bottler, beverage line). Speaks
production, not IT. Fears fines and line stoppages, tired of Excel-based
aggregation. Tone: plain, concrete, confident — "the line keeps running,
no fines, reports in two minutes". No buzzwords.

## Page structure (top to bottom)

1. **Hero:** one value sentence (working draft: «Маркировка „Честный знак" на
   вашей линии — без остановок и штрафов»), product visual (line station
   aggregation screen — the box-fill visual is our money shot), CTA + phone.
2. **Pains:** fines for marking errors, manual code reconciliation,
   Excel aggregation, line stoppages when "the marking system is down".
3. **How it works:** 3 steps — create a shift → scan & aggregate (offline OK)
   → download files for 1C / GIS MT.
4. **Features grid:** validation with duplicate control, 3-level aggregation
   with label printing, full-shift offline, multi-terminal shifts, label
   template editor, public API, RU/EN, light/dark.
5. **Product groups:** beer, water, soft drinks + "universal platform for
   Chestny ZNAK groups".
6. **Pricing:** placeholder block «от N ₽/мес за линию» — final numbers come
   later; design the block so tiers can be added without redesign.
7. **FAQ:** offline, hardware requirements (any ZPL/TSPL printer, USB/COM
   scanner), onboarding time, data safety.
8. **Contacts + demo request form:** name, company, phone — 3 fields max.

## Assets

Product screenshots come from briefs 03–04 mockups (no need to fake them).
Photography style if used: real production floors, not stock handshakes.

## Practical notes

- Form submissions and phone are the only conversions to optimize for.
- Page should be light and fast; no heavy animation. Subtle scroll reveals OK.
- SEO basics: semantic headings around «маркировка Честный знак»,
  «агрегация», «сериализация» keywords (copy will be provided/refined by us).
