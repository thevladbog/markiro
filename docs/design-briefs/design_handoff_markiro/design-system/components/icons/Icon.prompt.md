Иконка фирменного набора Маркиро (~45 знаков): навигация, статусы, оборудование (принтер, сканер, агент), агрегация (единица, короб, паллета, разагрегация), синхронизация/офлайн.

```jsx
<Icon name="scan" size={24} />
<Icon name="printer" size={32} color="var(--fg-2)" />
```

Статусные знаки (`ok-square`, `error-square`, `duplicate`, `sync`, `offline`, `alert`) всегда идут в паре с цветом и текстом — цвет никогда не единственный носитель смысла. В цеховом режиме иконки минимум 32px и всегда с подписью.
