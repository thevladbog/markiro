Выпадающий список на нативном select — работает с клавиатуры и на touch.

```jsx
<Select label="Продуктовая группа" options={["Пиво", "Вода", "Безалкогольные"]} value={g} onChange={setG} />
```

Для цеха предпочитайте крупные кнопки-варианты вместо селекта; Select mode="floor" — только для редких настроек.
