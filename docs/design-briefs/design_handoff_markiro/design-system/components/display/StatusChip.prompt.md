Статусный чип — главный носитель семантических цветов системы.

```jsx
<StatusChip kind="ok">Отправлено</StatusChip>
<StatusChip kind="duplicate" />
<StatusChip kind="syncing" mode="floor">Синхронизация</StatusChip>
```

Полные русские слова без сокращений («Синхронизация», не «Синхр.») — компонент рассчитан на длину RU-строк.
