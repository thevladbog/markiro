EmptyState + Skeleton — паттерны состояний каждой поверхности (пусто / загрузка / ошибка / офлайн).

```jsx
<EmptyState icon="report" title="Заданий пока нет" action={<Button>Создать задание</Button>}>
  Создайте первое задание — коды подтянутся из Честного ЗНАКА автоматически.
</EmptyState>
<Skeleton width={240} height={14} />
```

Каждое состояние заканчивается действием пользователя, не тупиком.
